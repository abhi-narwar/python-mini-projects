# 🐍 Python Mini Projects

This repository contains **3 beginner-friendly Python projects** designed to improve programming skills through practical implementation.  
Each project is small, simple, and demonstrates core Python concepts.

---

## 📂 Projects Included

1. **Employee Management System** – A simple Python application to manage employee records, including adding, updating, and viewing employee details.  
2. **Tic Tac Toe** – A classic two-player game implemented in Python, demonstrating loops, functions, and basic game logic.  
3. **Movie App** – A mini app to search and display movie details, showcasing Python basics and API integration.
